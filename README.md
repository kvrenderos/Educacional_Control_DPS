# Educacional_Control_DPS
Fase 1 del proyecto de cátedra.

## Descripción del proyecto

Educational Control es una aplicación móvil orientada a la gestión académica de instituciones educativas. 
El sistema permitirá registrar estudiantes, gestionar materias, controlar asistencia, registrar calificaciones y generar reportes académicos.

El objetivo del proyecto es facilitar el control y seguimiento del rendimiento académico de los estudiantes mediante una aplicación móvil.

## Tecnologías utilizadas

Frontend
- React Native

Backend
- Node.js
- Express

Base de datos
- MySQL

Control de versiones
- GitHub
